# CreditCardForm
Form for credit card | Форма для кредитной карты

![Site anim](/anim.gif)

> Данная форма послужит для использования в проектах, где необходим ввод данных кредитной карты, ну или как идея для своей формы :)
> This form is used for use in projects where you need to enter credit card information, or as an idea for your own form :)

## Стек | Stack
* JS
* Sass
* HTML

## Плагины | Plugins
* jQuery

## Сборщик проекта | Project builder
* gulp

### Инициализация проекта | Project setup 
```
npm install
```
### Компиляция и быстрая-перезагрузка в режиме разработки | Compiles and hot-reloads for development
```
gulp serve
```
### Компиляция проекта для продакшена |Compiles and minifies for production
```
gulp build
```
